# Newly Booked · B2B Funnel

A mobile-first B2B funnel for [Newly Booked](https://newlybooked.com) targeting medspa owners who want a true revenue-share growth partner.

## Pages

- `apply.html` — main landing with 5-step qualifying form, exit-intent popup, testimonials, FAQ
- `book-a-call.html` — calendar booking after qualification
- `thank-you.html` — post-booking confirmation with prep video
- `dq.html` — graceful disqualification (under $10K/mo or non-owner)
- `faq.html` — full FAQ
- `testimonials.html` — voices page

## Stack

Pure HTML/CSS/JS, no build step. Open `apply.html` directly or host any static file server.
